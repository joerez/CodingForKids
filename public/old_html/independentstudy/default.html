Independent Study

This is for the student that wishes to learn more about coding outside of class. I hope that the curriculum I reccomend here will serve as a great tool for continual study after the unit is over. Of course you are more than welcome to use these during the unit as well!

Code Academy

Code Academy is a great website for learning the basics of just about any Coding Language you can think of.

I reccomend using this as the place to start before trying out projects on your own.

Free Code Camp

FreeCodeCamp is a free bootcamp that operates online. I reccomend this over everything because after teaching the basics, it plunges you straight into making your own projects. FreeCodeCamp allows students to earn all kinds of certificates, and people have been able to get Coding jobs even before completing the program.

I reccomend that you start with FreeCodeCamp, and if you get stuck, to go to CodeAcademy and follow the course that will help you complete your FreeCodeCamp project.

As a student learning through FreeCodeCamp, you will be introduced to different tools, concepts, and programs that real developers use, and you'll be connected to a world-wide community of people going through the same thing as you!
