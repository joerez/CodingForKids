HTML Tag List

By Mr. Joe

My students have requested that I put together a definitive list of HTML tags that they can use for reference.

<!DOCTYPE html> - This should go on line one of your text-editor. This allows browsers to know how to interpret your code.

<html> - This should be the second element in your text-editor, it always goes right after <!DOCTYPE html> You should close it on the last line. Adittionally this let's the browser know to interpret all tags inside as HTML.

<head> - Your head tag should go underneath your html tag. I think of the head as the pages brain that links things together. You shouldn't put website content here.

<title> - Your title tag should be nested inside your head element. This tag changes what's displayed on browser Tabs.

<style> - The style tag should be nested inside of the head element. You can use CSS here.

<body> - This should go underneath your closing head tag. All site content should go here.

<h1> to <h6> - Big bold text. As the number gets larger, the text gets smaller. Search engines will search for your H1 tag, so make it relevent to your website.

<p> - This is your paragraph tag. This is used for creating paragraphs.

<br> - This is a break tag. It's used to make content go to the next line on your page. It doesn't need a closing tag.

<hr> - This is a horizontal line that goes all the way across the page.

<img src="URL"> - This is an image tag with an attribute that allows you to place images. URL should be replaced with your images location.

<div> - Divs allow you to divide content. Mostly useful in conjuction with CSS. Typically they will have a class or ID that you can style.

<button> - Creates a button.

<big> - Makes text bigger.

<strong> - Makes text bold.

<center> - Will center whatever you put inside. Sometimes I'll use this to avoid using CSS.

<iframe src="URL"> - Used to display another webpage on your webpage.

<ul> - Unordered list. For bullet points.

<ol> - Ordered list. For numbered points.

<li> - Used inside ordered and unordered lists. Creates either a bullet point or a number.

<a href="URL"> - Creates a clickable link.

I will add more as we continue our unit. If you're trying to do something without a tag listed that's here, there are all sorts of other resources that you can use online! I'll be continually adding onto this page as we go.
