Codepen

Codepen is an excellent web-application that allows you to enter HTML, CSS, and Javascript, and see your live web-page. Many students use Codepen to share their code, get help from other developers online, and build prototypes. Additionally, it acts as a great way to host your creations.

Visit Codepen

Different CSS Libraries

  • Bootstrap - Perfect for creating navbars, buttons, and alligning content.
  • jQuery - jQuery is a fast, small, and feature-rich JavaScript library.
  • Animate.css - Add awesome animations to your web-page.
  • Wow Animations - Makes animations only occur once you scroll to the element.
  • Smooth Scroll - Javascript code that smooth scrolls to element IDs.

W3.org

W3.org is a website you'll constantly come across as you learn how to code. When loooking up different html tags, css properties, attributes, or Javascript methods, you'll find that W3.org is an excellent referencing tool. I still use it all the time, and you should too. In my opinion, W3.org is the difinitive resource site for learning how to code.
